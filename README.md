## Usage

You need compiled `libbncsutil.dylib` or `libbncsutil.so` or `bncsutil.dll` in directory root.

And setup server credentials in `config.json`

```bash
git clone https://github.com/w3gh/ghost.js.git ghost
cd ghost
npm i
npm start
```
